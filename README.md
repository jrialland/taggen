# taggen
A dumb java library for generating random names


[![Build Status](https://travis-ci.org/jrialland/taggen.svg)](https://travis-ci.org/jrialland/taggen)

Invocation :
------------

```java
	import net.jr.taggen.TagGenerator;
	
	[...]
	
	TagGenerator.newCamelcaseTag();
```
